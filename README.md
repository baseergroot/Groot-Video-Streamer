# Groot Video Streamer

Stream and download YouTube videos with a FastAPI backend and a Next.js frontend.

## For Users (Run the App)

1. Start the backend: see `fastapi/README.md` (dev-only).
2. Start the frontend: see `nextjs/README.md`.
3. Open the frontend in your browser.

## For Developers

- Backend: `fastapi/README.md`
- Frontend: `nextjs/README.md`

## Project Structure

- `fastapi/` — API server (FastAPI)
- `nextjs/` — UI client (Next.js)
