# Next.js Frontend

Use this UI to stream, view info, and download videos.

## For Users (Run Locally)

1. Install dependencies:
```bash
npm install
```

2. Start the dev server:
```bash
npm run dev
```

3. Open `http://localhost:3000`.

## For Developers

### Configure API URL
Set the backend base URL if needed:

- Create `.env` in this folder
- Add:
```bash
API_URL=http://localhost:8000
```

### Build
```bash
npm run build
npm run start
```
